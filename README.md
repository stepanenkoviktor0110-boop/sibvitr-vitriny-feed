# Сибвитрина — курированный фид витрин для B24U

30 разнотипных витрин из каталога sibvitr.ru (WooCommerce Store API).
Сборка: `npm run build` → `public/feed.xml`. Авто-ребилд каждые 6ч (GitHub Actions), раздача через Pages.
Фид витрин = зона продуктовой точности бота (U2, карточки с ценой); остальной каталог — навигация по разделам (U1).
